 🏦 Bank Management System in C++

A simple console-based Bank Management System built using C++ that demonstrates Object-Oriented Programming (OOP) and File Handling concepts.

 📌 Features

- 🔐 Admin Login Authentication
- 👤 Create New Bank Accounts
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- ❌ Close Bank Account
- 🧾 Transaction History Tracking
- 💾 File Handling for Permanent Data Storage

 🛠️ Technologies Used

- Language: C++
- Concepts:
  - Object-Oriented Programming (OOP)
  - File Handling
  - Vectors (STL)
  - Console-based UI
